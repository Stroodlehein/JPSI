Core Formulas

// Refinery Spot average
RS = mean(Tanaka, Tokuriki, Ishifuku)

// Dealer Bid average
DB = mean(Dealer1, Dealer2, Dealer3)

// Composite Index
JPSI = (RS × wRS) + (DB × wDB) + (SR × wSR)

// Street Retail premium
Premium = (SR / RS) − 1

// Mercari per-item price
¥/g = Sold Price (¥) ÷ 31.1035
Disclaimer: JPSI is an informational index only. Street Retail (SR) depends heavily on item mix (coins vs art vs numismatic). For the cleanest signal, restrict SR inputs to common bullion .999 sold listings only.
