Core Formulas

// Refinery Spot average
RS = mean(Tanaka, Tokuriki, Ishifuku)

// Dealer Bid average
DB = mean(Nanboya, Daikichi, ...)

// Composite Index
JPSI = (RS × wRS) + (DB × wDB) + (SR × wSR)

// Street Retail premium
Premium = (SR / RS) − 1

// Mercari per-item price
¥/g = Sold Price (¥) ÷ 31.1035
Disclaimer: JPSI is an informational index only. RS and DB prices are fetched daily via GitHub Actions from public sources. Street Retail (SR) must be entered manually from Mercari sold listings. Restrict SR to common .999 bullion (1oz) sold listings for the cleanest signal.
