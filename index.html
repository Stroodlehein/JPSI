What JPSI measures
RS (Refinery Spot): upstream physical buyback (Tanaka + Tokuriki + Ishifuku proxy).
DB (Dealer/Pawn Bid): real exit liquidity (Nanboya SV1000 + Daikichi SV1000).
SR (Street Retail): what people pay on Mercari (sold listings).
JPSI: weighted composite — “true physical” in JPY.
Core formulas
RS = mean(Tanaka_buy, Tokuriki_buy, Ishifuku_buy) DB = mean(Nanboya_SV1000, Daikichi_SV1000) JPSI = (RS*wRS) + (DB*wDB) + (SR*wSR) Premium_vs_RS = (SR / RS) - 1

Signals
SR ↑ while RS flat → retail hoarding / premium expansion
DB ↑ before RS → scrap shortage / dealer competition
SR listings vanish → “panic / keep it” phase
Premium > 30% → tight physical conditions (Japan retail)
