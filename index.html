What JPSI measures
RS (Refinery Spot): upstream physical buyback in Japan (e.g., Tanaka/Tokuriki/Ishifuku).
DB (Dealer/Pawn Bid): real exit liquidity for scrap & common items.
SR (Street Retail): what people are paying on Mercari (sold listings).
JPSI: weighted composite — “true physical” in JPY.
Core formulas
RS = mean(Tanaka_buy, Tokuriki_buy, Ishifuku_buy) DB = mean(Dealer1, Dealer2, Dealer3) JPSI = (RS*wRS) + (DB*wDB) + (SR*wSR) Premium_vs_RS = (SR / RS) - 1
MSPI (Mercari) per-item
Adjusted_Silver_g = Total_Weight_g * Purity Price_per_g = Price_Yen / Adjusted_Silver_g For bullion coins: Price_per_g = Price_Yen / 31.1035
Signals
SR ↑ while RS flat → retail hoarding / premium expansion
DB ↑ before RS → scrap shortage / dealer competition
SR listings vanish → “panic / keep it” phase
Premium > 30% → tight physical conditions (Japan retail)
